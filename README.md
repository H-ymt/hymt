Personal Website built with Next.js and Cloudflare Workers 🚀

## Stack

- Next.js
- TypeScript
- CSS Modules
- Cloudflare Workers
- Cloudflare R2/KV

## Deploy

```bash
pnpm deploy
```

## Preview

```bash
pnpm preview
```

## Fetch Knowledge

```bash
pnpm fetch-knowledge
```

## Sync KV

```bash
pnpm sync-kv
```

## Upload Assets

```bash
pnpm upload-assets
```
